## 📊 Project Overview

The **Coffee Shop Sales Dashboard** is an interactive data analytics project built using **Microsoft Excel**.  
This project transforms raw coffee shop sales data into **meaningful business insights** through data cleaning, pivot tables, and dynamic visualizations.

The dashboard helps analyze **sales performance, product demand, and revenue trends**, allowing businesses to make **data-driven decisions** efficiently.
